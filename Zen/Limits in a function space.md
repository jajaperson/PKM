[[Function space]]
# Limits in a function space
The [[Convergence]] of functions is a concept underlying [[Taylor series]] and other series of functions.
The main definition of **convergence** of a sequence of functions is given by [[Uniform convergence]] which uses the [[Supremum metric]].
This has much nicer properties than pointwise convergence.
