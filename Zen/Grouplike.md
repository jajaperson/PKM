# Grouplike in a $\mathcal{K}$-monoid

In a [[R-comonoid]] $\mathcal{A}$, the comultiplication can almost never be the diagonal map, since this is not linear.
It can however act diagonally on a subset of elements.
A <dfn>grouplike</dfn> of $\mathcal{A}$ is a _nonzero_ element $x \in \mathcal{A}$ such that $\Delta x = x \otimes x$. #m/def/ralg/hopf 

![[grouplike-string.svg#invert|c]]

The set of all grouplikes in $\mathcal{A}$ is denoted $\opn{Gr}\mathcal{A}$.
