[[Functor]]
# Constant functor

A **constant functor** $AT : \cat C \to \cat D$ is a [[functor]] that factorizes via the [[Trivial category]],
and therefore maps every object in $\cat C$ to an object $A \in \cat D$ and every morphism to $\id_{A}$.
