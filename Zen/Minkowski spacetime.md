# Minkowski spacetime
**Minkowski spacetime** is the formal name for the combination of euclidean space and time into a four-dimensional manifold,
for which the [[Spacetime interval]] $(\Delta s)^2$ is an invariant metric.

## Subdivision
Since $(\Delta s)^2$ is invariant for all [[Inertial frame of reference|inertial frames]],
it can be used to subdivide all of spacetime in relation to one event (taken as the origin). These regions are as follows:

1. If $(\Delta s)^2 < 0$[^conv], i.e. $(c \Delta t)^2 > (\Delta r)^2$,
   the event is **time-like** separated from the origin.
   This means ^time-like
   - In all reference frames the events have the same order and never occur simultaneously,
     i.e. they may be causally related.
   - There exists a reference frame where the events occur in the same place.
2.  If $(\Delta s)^2 > 0$[^conv], i.e. $(\Delta r)^2 > (c \Delta t)^2$,
   the event is **space-like** separated from the origin.
   This means ^space-like
     - In all reference frames the events have spacial separation
       i.e. they may be causally related.
     - There exists a reference frame where the events occur simultaneously
       (see [[Relativity of simultaneity]]).
3. If $(\Delta s)^2 = 0$, i.e. $(\Delta r)^2 = (c \Delta t)^2$, the event is **light-like** separated.
   This means ^light-like
   - It is possible a photon to travel in a straight line between the events
   - No object with mass can be present at both events.

[^conv]: The $<$ and $>$ are reversed when the negative space convention is used (as is the case for [[PHYS1002]]). See the footnote in [[Spacetime interval]] for more discussion of convention.
