# Comonoid morphism

Given [[Comonoid object|comonoids]] $C,C'$ in $\cat C$, a morphism $f : C \to C'$ is a **comonoid morphism** iff $\op f: C' \to C$ is a [[Monoid morphism]] in $\op{\cat C}$. #m/def/cat
