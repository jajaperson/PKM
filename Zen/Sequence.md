# Sequence
A sequence is an  (often infinite) ordered collection in which repeats are allowed.
Alternatively, a sequence $(a_n)_{n=1}^\infty$ may be thought of as a function $f: \mathbb N \to \{a_{n}\} : n \mapsto a_{n}$.
If $f$ is a bijection, the sequence induces a [[Totally ordered set]] of $\{ a_{n} \}$ with an initial object.
Similarly, any countable set can be given a total order by a bijective sequence.
By skipping some terms of a sequence one gets a [[Subsequence]].
