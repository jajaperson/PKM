# Estimator
In probability theory, an **estimator** is a [[statistic]] (a type of [[real random variable]])
which **estimates** some underlying parameter 
of some [[Random sample|independent and identically distributed]] random variables.
A particular result of an estimator,
yielded by applying the statistic to a particular [[random sample]],
is called an **estimate**.

## Examples

- [[𝜇-estimator|Sample mean]], an example of a [[Sample moment]].
- [[Sample variance]]
- [[Sample skewness]]
- [[Sample excess kurtosis]]
