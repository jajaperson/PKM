# Objects as functors

Let $\cat C$ be a [[category]].
It is often useful to view an object $X \in \Ob \cat C$ as a [[functor]] (or [[Commutative diagram|diagram]]) from the single-object [[trivial category]]
$$
\begin{align*}
\underline 1 &\to \cat C \\
0 &\mapsto X
\end{align*}
$$
See also [[objects as identities]].
