# Bendixson-Dulac criterion

Consider the dynamical system $\dot{\mathbf{x}} = f(\mathbf{x})$
Let $R \sube \mathbb{R}^2$ be a simply-connected set,
and $\varphi \in \Man_{1}(R,\mathbb{R})$.
If
$$
\begin{align*}
\opn{div}(Bf)
\end{align*}
$$
has the same sign [[Almost everywhere]]
then the system has no nonconstant periodic solutions.
