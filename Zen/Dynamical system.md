# Dynamical system

A **dynamical system** is a set of variables (a vector in phase space) whose values change with time according to some predetermined rules (the equations of motion). #m/def/dynamics 
Two main forms are the [[Differential system]] (continuous real time)
$$
\begin{align*}
\dot{\mathbf{x}} = f(\mathbf{x})
\end{align*}
$$
and the [[Iterated map]] (discrete time)
$$
\begin{align*}
\mathbf{x}_{n+1} = f(\mathbf{x}_{n})
\end{align*}
$$
The fundamental difference is that in the differential case $f$ describes the rate of change
(hence it is sometimes called a velocity field)
whereas for an iterated map it describes the change itself.
