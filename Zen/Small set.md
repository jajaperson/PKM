# Small set

The precise meaning of **small set** depends on the choice of [[Foundation of mathematics|foundation]] being used.
In [[ZFC]] and its [[conservative extension|conservative extensions]] (e.g. [[NBG]]),
the term merely emphasizes that a [[set]] is not a [[proper class]].
In foundations with universes such as [[TG]], a set $x$ is **small** with respect to a universe $U$ if $x \in U$. #m/def/set/tg
In type theory, it is an [[h-set]] which is also a [[small type]].
