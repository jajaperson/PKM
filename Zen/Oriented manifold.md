# Oriented manifold

Let $(M, \mathscr{A})$ be a $C^\alpha$-[[manifold]].
An [[atlas]] $\mathscr{B} \sube \mathscr{A}$ is called **oriented** iff all [[Transition map|transition maps]] have positive [[Jacobian matrix|Jacobian determinant]].
An **orientation** on $M$ is the choice of a maximal oriented atlas. #m/def/geo/diff 
If an orientation exists we say that $M$ is **orientable**.

- If $M$ is [[Semi-Riemannian manifold|semi-Riemannian]], we can use orientation to define a [[Riemannian volume form]] as well as the [[Hodge star]].

## Properties

- [[Volume form#^P2|A manifold is orientable iff there exists a nonvanishing volume form]]
