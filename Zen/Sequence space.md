# Sequence space

A **sequence space** is a set of [[sequence|sequences]] in some fixed set.
Thus a sequence space is a special case of a [[function space]] where the domain is the natural numbers $\mathbb{N}$.
If the underlying set is a [[field]],
then the sequence space naturally carries the structure of a [[vector space]].
