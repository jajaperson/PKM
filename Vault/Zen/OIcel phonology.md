## Rules

- geminate plosive preäspiration `PP -> ʰP`
- f-voicing `f -> v / {_[+voice] [+voice]_}`
- g-frication `g -> ɣ / {[+syll]_[+syll] _[-voice]}`
- v-approximation `v -> ʋ / [+syll]_[+syll]`
- [[OIcel u-umlaut]] `a -> ǫ / _Cu`
-  `{n l s}r# -> {n l s}r#`
