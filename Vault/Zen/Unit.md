# Unit

Let $R$ be a [[ring]]. 
A **unit** is an element $a \in R$ with a two-sided multiplicative inverse, #m/def/ring 
i.e. there exists a (provably unique) element $a^{-1} \in R$ such that $a^{-1} a = aa^{-1} = 1$.
The set of all units is the [[group of units]].
