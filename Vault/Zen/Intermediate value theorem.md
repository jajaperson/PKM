# Intermediate value theorem

Let $I = [a,b]$ and $f \in \Top(I,\mathbb{R})$ be a [[Continuity|continuous]] function.
Then $f(I)$ is a closed interval containing $[\min \{ f(a),f(b) \}, \max\{ f(a),f(b) \}].$ #m/thm/anal
