# Simple module

A [[module]] $M \in \lMod R$ is **simple** or **irreducible** iff it has no nontrivial proper [[Submodule]]. #m/def/module 

## Properties

- [[Schur's lemma]] states that any nonzero homomorphism between simple modules is an isomorphism.
