# Matrix calculus

This Zettel is a collection of identities for infinitesimal calculus with matrices.

## Invertible matrix

Let $\mathbf{M} : \mathbb{R} \to \GL_{n}(\mathbb{C}) : t \mapsto M(t)$ be a $C^1$-map.

1. $$
  \frac{ \partial }{ \partial t }  \ln \abs{\det \mathbf{M}} = \Tr \left( \mathbf{M}^{-1} \frac{ \partial \mathbf{M} }{ \partial t }  \right)
