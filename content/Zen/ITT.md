[[Type theory MOC]]
# ITT

[[Per Martin-Löf|Per Martin-Löf's]] **Inductive Type Theory** or **ITT** is a [[dependent type theory]] with [[judgemental equality]]
along with the following types along with their standard inference rules: #m/def/type

1. [[Π type]]
2. [[Σ type]]
3. [[Coproduct type]]
4. [[Empty type]]
5. [[Unit type]]
6. [[Identity type]]
