# International Keyboard Cheatsheet
The international keyboard on MacOS allows the typing of several languages,
as well as phonetic transcriptions,
using one keyboard layout.

## Diacritics
- **ä** is <kbd>⌥u</kbd>
- **å** is <kbd>⌥k</kbd>
- **ā** is <kbd>⌥a</kbd>
- **ʉ** is <kbd>⌥l</kbd>

## Phonetic
- **ə** is <kbd>⌥⇧; a</kbd>
- **ɔ** is <kbd>⌥⇧; c</kbd>
- **ʃ** <kbd>⌥⇧. s</kbd>
- **œ** is <kbd>⌥q</kbd>
- **æ** is <kbd>⌥'</kbd>

## Delimiters
- **‹** is <kbd>⌥⇧3</kbd>
- **›** is <kbd>⌥⇧4</kbd>
