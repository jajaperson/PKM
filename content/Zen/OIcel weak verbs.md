[[OIcel conjugation]]
# OIcel weak verbs

As with the [[PGmc weak verbs]] they descend from,
the distinguishing feature of the OIcel weak verb is a dental suffix in the past tense.
However there is variation in the dental suffix used:

- `ð / {V r f [+voice +fricative]}_`
- `d / [+voice -fricative]_`
- `t / {l [-voice]}_`

## Subclasses

The following classification is used in Norröne Grammatik im Überblick

| Haugen                | Traditional | PNGmc                      |
| --------------------- | ----------- | -------------------------- |
| [[OIcel telja-class]] | I.a         | <em class="recon">ja</em>  |
| [[OIcel dǿma-class]]  | I.b         | <em class="recon">ija</em> |
| [[OIcel kasta-class]] | II          | <em class="recon">ō</em>   |
| [[OIcel duga-class]]  | III         | <em class="ling">ē</em>    |

## Basic paradigm

As with [[PGmc weak verbs#Common features]], all OIcel weak verbs take identical^[up to dental suffix variation, see above] endings in the finite past,
however variation of the stem in these forms did vary.

| pret. | indic.                      | subj.                       |
| ----- | --------------------------- | --------------------------- |
| 1sg   | <em class="ling">-d-a</em>  | <em class="ling">-d-a</em>  |
| 2sg   | <em class="ling">-d-ir</em> | <em class="ling">-d-ir</em> |
| 3sg   | <em class="ling">-d-i</em>  | <em class="ling">-d-i</em>  |
| 1pl   | <em class="ling">-d-um</em> | <em class="ling">-d-im</em> |
| 2pl   | <em class="ling">-d-uð</em> | <em class="ling">-d-ið</em> |
| 3pl   | <em class="ling">-d-u</em>  | <em class="ling">-d-i</em>  |

The present varied more widely, but may be summarised as

| pres. | indic.                          | subj.                     |
| ----- | ------------------------------- | ------------------------- |
| 1sg   | <em class="ling">-∅/i/a</em>    | <em class="ling">-a</em>  |
| 2sg   | <em class="ling">-(∅/i/a)r</em> | <em class="ling">-ir</em> |
| 3sg   | <em class="ling">-(∅/i/a)r</em> | <em class="ling">-i</em>  |
| 1pl   | <em class="ling">-(j/∅)-um</em> | <em class="ling">-im</em> |
| 2pl   | <em class="ling">-ið</em>       | <em class="ling">-ið</em> |
| 3pl   | <em class="ling">-(j/∅)-a</em>  | <em class="ling">-i</em>  |

Note that the [[OIcel subjunctive]] forms are common to all verbs.
