# Boundary value eigenvalue problem
A **boundary value eigenvalue problem** involves finding the various eigenfunctions that can satisfy boundary conditions. For example[^2017]
$$
\begin{align*}
\frac{dy}{dx} + \lambda y = 0
\end{align*}
$$
with boundary conditions
$$
\begin{align*}
y(0) = 0, \quad y(\pi) = 0
\end{align*}
$$
has eigenfunctions of the form
$$
\begin{align*}
y=\sin(n x)
\end{align*}
$$
with eigenvalues $\lambda = n^2$.

[^2017]: 2017\. [[Sources/@boyceElementaryDifferentialEquations2017|Elementary differential equations and boundary value problems]], pp. 466ff.
